# 3단계 로또 2등

### 클래스 정의
 - `로또` 클래스를 생성한다. _<b>Lotto</b>_
 - `전체 로또` 클래스를 생성한다.  _<b>Lottos</b>_
 - `로또가격` 클래스를 생성한다.  _<b>LottoPrice</b>_
 - `당첨통계` 클래스를 생성한다. _<b>ResultView</b>_
 - `로또 입력` 클래스를 생성한다  _<b>InputView</b>_
 - `컨트롤러` 클래스를 생성한다. _<b>Controller</b>_
 - `로또 순위` ENUM을 생성한다. <b>Rank</b>
 
### 클래스 별 기능 생각
 - _<b>Lotto</b>_
    1. 자신만의 번호 6자리가 있다.
    2. 가격이 정해져있다.
 - _<b>Lotto</b>_   
    1. 입력받은 로또의 개수 만큼 로또를 가지고있는다.
 - _<b>LottoPrice</b>_
    1. 로또 가격을 입력 받는다.
 - _<b>ResultView</b>_
    1. 오구사항의 출력기능만 한다.
 - _<b>InputView</b>_
    1. 요구사항의 입력기능만 한다.
 - _<b>Rank</b>_
    1. 로또 순위에 대한 금액이 있다.
    2. 일치하는 갯수를 매칭시킨다.         
  - _<b>LottoService</b>_
     1. 핵심 로직이 담겨있다.
    
### 메시지를 먼저 생각하고, 객체를 정해보자.

 - 돈을 입력받는다.
 - 로또 하나 생성한다.
 - 받은 금액만큼 생성한다.
 - 
 - 지난 당첨 로또번호 입력해
 - 당첨번호에 보너스볼 하나더 입력
 - 각 로또번호마다 몇개 일치하는지 알려줘
