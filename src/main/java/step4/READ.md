# 🚀 4단계 - 로또 (수동)

### 
 기능 구현 사항
 - 로또
   - 로또 번호
     - 1 ~ 45까지 숫자만 가능하다.
   - 자릿 수
     - 6자리만 가능
     - 중복을 허용할 수 없다.
   - 자동
     - 입력받은 숫자에 따라 자동으로 로또 번호를 생성한다.
   - 수동
      - 입력받은 숫자에 따라 수동으로 생성한다.
   - 보너스
     - 기존의 로또번호와 중복되면 안된다.
     - 5개를 맞친 후 보너스가 맞으면 2등.
     - 아니면 3등
   - 당첨
     - 지난 주 당첨되었던 로또를 입력한다.
     - 지난 주와 생성된 로또들과 비교한다.
   - 통계
     - 지난 주 로또 번호와 비교된 것들에 대한 통계를 낸다.
